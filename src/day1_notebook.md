# C Compiler

## 整数 1 個のコンパイル

- コンパイラの概要
  c 言語 -> (IL: LLVM の中間言語) -> .s (assembly) ファイル -> .o (object) ファイル -> リンク -> バイナリ

## トークナイズ

"1 + 1" => ["1", "+", "1"]

- 連結リスト
- 前のトークンが次のトークンの情報をもつ
- 例: トークン配列

```C
  ["1", "+", "1"]
  "1".next -> "+".next -> "1"
  // ポインター
  Token.next -> Token
```

- 構文解析を行うもの
- 空白文字を処理したい
